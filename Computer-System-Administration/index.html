    Starting

    • set almost every default.

    Sudo

    Install

    1. switch to root by using su -
    2. install package sudo 
      pkg install sudo

    Allowing user to execute sudo

    1. use visudo to edit sudoer file 
      ##
## User privilege specification
##
root ALL=(ALL) ALL
[user name] ALL=(ALL) ALL
    2. save and logout
    3. type sudo whoami to check if it works.

    Excute sudo without password

    1. use visudo to edit sudoer file 
      ##
## User privilege specification
##
root ALL=(ALL) ALL
[user name] ALL=(ALL) NOPASSWD:ALL
    2. save and logout

    Add user

    Update FreeBSD

    1. fetch new version 
      sudo freebsd-update fetch
    2. install fetch 
      sudo freebsd-update install
    3. reboot and check version 
      sudo shutdown -r now

freebsd-version

    HW1

    Enable sshd

    1. fetch ssh public key
    2. install to /home/[username]/.ssh/authorized_keys
      • make .ssh and authorized_keys yourself if they doesn’t exist

    Install WireGuard

    It’s a VPN server, will be use by OJ.

    1. install it though pkg

      pkg install wireguard

    2. put configuration to /usr/local/etc/wireguard

      • download from OJ-Debug_tools

    3. use wq-quick to start/stop the connection

      wg-quick [up/down] [conf]
