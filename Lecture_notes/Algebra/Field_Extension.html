Field Extension

Definition

A field $E$ is an extension field of a field $F$ if $F \leq E$.

Definition

Let $F \leq E$. If there exists $\alpha \in E$ such that $E = F(\alpha)$, then $E = F(\alpha)$ is a simple extension of $F$.

Definition

An ekement $\alpha$ of an extension field $E$ of a field $F$ is algebraic over $F$ if $f(\alpha) = 0$ for some nonzero polynomial $f(x) \in F[x]$.

If such polynomial does not exist, then $\alpha$ is transcendental over $F$.

Definition

An element $\alpha \in \Bbb{C}$ is an algebraic number if $\alpha$ is algebraic over $\Bbb{Q}$.

And it’s a transendentental number if it is transendental over $\Bbb{Q}$.

Irreducible polynomial for $\alpha$ over $F$

Assume that $F \leq E$, and $\alpha \in E$ is algebraic over $F$.

Let $I_\alpha = { f(x) \in F[x] \mid f(\alpha) = 0}$ be a subset of $F[x]$.

Lemma

The polynomial $p_\alpha(x)$ is irreducible where $p_\alpha(x)$ is the nonzero polynomial of minimal degree in $I_\alpha$.

**Proof**

Definition

Let $F$ be a field.

A polynomial is $F[x]$ with leading coefficient $1$ is called a monic polynomial.

Definition

The unique monic irreducible polynomial $p(x)$ in $I_\alpha$ is called the irreducible polynomial for $\alpha$ over $F$, and is denoted by ${\rm Irr}(\alpha, F)$.

The degree of ${\rm Irr}(\alpha, F)$ is the degree of $\alpha$ over $F$, and is denoted by ${\rm deg}(\alpha, F)$.

Simple Algebraic Extension

Theorem

Let $F(\alpha)$ be an algebraic extension field of $F$. Then $F(\alpha) = F[\alpha]$.

**Proof**

Theorem

Let $F(\alpha)$ be an algebraic extension field of $F$. Then any $\beta \in F(\alpha)$ can be uniquely expressed in the form
$$\beta = b_0 + b_1 \alpha + \cdots + b_{n-1} \alpha^{n-1}$$
where $n = {\rm deg}(\alpha, F)$

**Proof**

Problem

Let $\alpha$ be a zero of $f(x) = x^3 + x + 1$ in $\Bbb{Q}[x]$.

Write $\beta = (2\alpha^2 + 3\alpha -1)/(\alpha^2 - \alpha + 1)$ in the form $a_0 + a_1\alpha + a_2 \alpha^2$.

Use the Euclidean algorithm to find polynomials $g(x), h(x)$ such that $$(x^3 + x + 1)g(x) + (x^2 - x + 1)h(x) = 1$$
Then when substitute $\alpha$,
$$\begin{align}
(\alpha^3 + \alpha + 1)g(x) + (\alpha^2 - \alpha + 1)h(x) &= 1 \
(\alpha^2 - \alpha + 1)h(x) &= 1 \
h(x) &= (\alpha^2 - \alpha + 1)^{-1} \
\end{align}$$
And we can simplely multiple $h(x)$ by $(2\alpha^2 + 3\alpha -1)$ to get the answer.

Start with finding $h(x)$.
$$\begin{align}
x^3 + x + 1 &= (x + 1)(x^2 -x + 1) + x \
x^2 - x + 1 &= (x - 1)(x) + 1 \
\end{align}$$
Then we get
$$\begin{align}
1 &= (x^2 - x + 1) - (x - 1)x \
&= (x^2 - x + 1) - (x - 1)\left[(x^3 + x + 1) - (x + 1)(x^2 - x + 1)\right] \
&= x^2(x^2 - x + 1) - (x - 1)(x^3 + x + 1) \
\end{align}$$
Therefore $h(x) = x^2$

Thus,
$$\begin{align}
\dfrac{2\alpha^2 + 3\alpha -1}{\alpha^2 - \alpha + 1} &= 2\alpha^4 + 3\alpha^3 - \alpha^2 \
&= (2\alpha + 3)(\alpha^3 + \alpha + 1) + (3\alpha^2 + 5\alpha + 3) \
&= -3\alpha^2 -5\alpha - 3 \
\end{align}$$
