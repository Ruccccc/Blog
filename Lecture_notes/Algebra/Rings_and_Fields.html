Rings

Definition

A ring $\langle R, +, \cdot \rangle$ is a set $R$ with two binary operations $+$ and $\cdot$, called addition and multiplication, such that the following axioms are satisfied:

  1. $\langle R, + \rangle$ is an abelian group.
  2. Multiplication is associative. That is $a(bc) = (ab)c$ for all $a, b, c \in R$.
  3. For all $a, b, c \in R$, the left distributive law and right distributive law holds. That is $a \cdot (b+c) = (a \cdot b) + (a \cdot c)$ and $(a + b) \cdot c = (a \cdot c) + (b \cdot c)$.
**Examples of Ring**
  1. The set $\mathbb{Z}, \mathbb{Q}, \mathbb{R}, \mathbb{C}$ are all rings with usual addition and multiplication.
  2. The set $\mathbb{Z}_n$ of all residue classes modulo $n$ is a ring with $\bar a + \bar b = \overline{a + b}$ and $\bar a \cdot \bar b = \overline{ab}$.
  3. The set $M_n(\mathbb{R})$ of all $n \times n$ matrices is a ring.
  4. Let $R_1$ and $R_2$ be rings. Define $+$ and $\cdot$ on $R_1 \times R_2$ by
    $$(a_1, b_1) + (a_2, b_2) = (a_1 + a_2, b_1 + b_2)$$ and
    $$(a_1, b_1) \cdot (a_2, b_2) = (a_1 \cdot a_2, b_1 \cdot b_2)$$
    Then $R_1 \times R_2$ is a ring, called the direct product of $R_1$ and $R_2$.
  5. Let ${\rm End}(V)$ be the set of all linear transformations from $V$ to $V$. Let addition and multiplication be defined by
    $$f + g : v \mapsto f(v) + g(v)$$$$f \circ g : v \mapsto f(g(v))$$
    Then $\langle {\rm End}(V), +, \circ \rangle$ is a ring, called the endomorphism ring of $V$.
**Show that $R$ is a ring**

Let $R$ be a set with $+$, $\ast$ be addition and multiplication.

  1. Show $\langle R, + \rangle$ is a abelian group
    1. $+$ is associative since $R$ is a ring and $+$ is induced from $R$.
    2. $\exists e \in R$ such that $\forall a \in R$, $a + e = e + a = a$
    3. $\forall a \in R, \exists -a \in R$ such that $a + (-a) = (-a) + a = 0$
    4. $\forall a, b \in R$, $a + b = b + a$
  2. Show multiplication is associative.
    $\forall a, b, c \in R$, $a(bc) = (ab)c$
  3. Show that left and right distributive law holds.
    $\forall a, b, c \in R$, $a(b + c) = ab + ac$ and $(a + b) c = ac + bc$

Basic properties of Rings

Theorem

Let $R$ be a ring. For any $a, b \in R$, we have

  1. $0_R a = a 0_R = 0_R$
  2. $a (-b) = (-a) b = -ab$
  3. $(-a)(-b) = ab$
**Proof**

  1. Since $0_R + 0_R = 0_R$, we have $(0_R + 0_R)a = 0_R a$. By the distributive law, we have
    $$0_R a + 0_R a = 0_R a$$
    By cancellation law,
    $$0_R a = 0_R$$
    By similar method, we can also prove $a0_R = 0_R$.

  2. Since $b + (-b) = 0_R$, we have $$ab + a(-b) = a 0_R = 0_R$$
    Then we can say that $ab$ is the inverse element of $a(-b)$. That is
    $$a(-b) = -ab$$
    We can prove $(-a)b = -ab$ in similar way.

  3. Applying the result of (2) twice, we can get
    $$(-a)(-b) = -(a(-b)) = -(-ab)$$
    It means that $-ab$ is the inverse element of $(-a)(-b)$.

    But we know that the additiveinverse of $-ab$ is $ab$.

    Thus, by the uniqueness of additive inverse, we conclude that $(-a)(-b) = ab$.

Multiplication of Rings

Definition

  1. A ring in which the multiplication is commutative is a commutative ring.
  2. If a element $a$ of $R$ satisfies $ra = ar = r$ for all $r \in R$, then $a$ is the multiplicative identity or the unity, and is denoted by $1$ or $1_R$.
  3. A multiplicative inverse of an element $a$ in a ring with unity is an element $a^{-1}$ of $R$ such that $a^{-1}a = a a^{-1} = 1$.

Definition

Let $R$ be a ring with unity $1 \neq 0$.

  1. An element $u$ of $R$ is a unit if it has a multiplicative inverse.
  2. If every nonzero element of $R$ is a unit, then $R$ is a division ring or skew field

Remark

  1. The indentity element of a ring is unique if it exist.
  2. The inverse element of a element is unique if it exist.
  3. For a ring with unity, the set of units forms a multiplicative group, denoted $R^\times$, called the group of units.

Field

Defintion

A commutative division ring is field.

Subring and Subfield

Definition

A subset $S$ of a ring $R$ is a subring of $R$ is $S$ if s is ring under the induced addition and multiplication from $R$. We denote this relation by $S < R$.

A subfield is similarly defined.

Criterion of Subring

A subset $S$ of ring $R$ is a subring of $\langle R, +, \cdot \rangle$ if

  1. $\langle S, + \rangle$ is a subgroup of $\langle R, + \rangle$.
  2. $S$ is close under multiplication of $R$.

Theorem - Subring test

A subset $S$ of ring $R$ is a subring of $\langle R, +, \cdot \rangle$ if

  1. $S$ in non-empty.
  2. For all $a, b \in S$, $a - b \in S$.
  3. For all $a, b \in S$, $a \cdot b \in S$.
**Proof**

First show that $\langle S, + \rangle$ is a subgroup of $\langle R, + \rangle$.

  1. Let $0_R$ be the identity element of $R$, $x \in S$.
    By second condition, we know that $x - x = 0_R \in S$.

    Therefore, $0_R \in S$.
  2. Take $a = 0_R$, $\forall b \in S$, by second condition, we can get $a - b = 0_R - b = -b \in R$

    Thus, we know that every element has a corresponding inverse element in $S$.
  3. For all $a, b \in S$, we can take $x = a, y = -b$ since we had show that $-y$ is in $S$ above.

    Then $a + b = x - y \in S$ is given by second condition.

    Thus, we know that $S$ is close.

And we already know that $S$ is closed under multiplication by the third condition.
Therefore, $S$ is a subring of $R$.

Homomorphism

Definition

For rings $R$ and $R^\prime$, a funtion $\phi : R \to R^\prime$ is a (ring) homomorphism if

  1. $\phi(a + b) = \phi(a) + \phi(b)$
  2. $\phi(ab) = \phi(a)\phi(b)$
    for all $a, b \in R$.

The kernel of $\phi$ is the set ${\rm ker}(\phi) = { a \in R \mid \phi(a) = 0 }$.

Theorem

Let $\phi: R \to R^\prime$ be a ring homomorphism. Then $\phi®$ is a subring of $R^\prime$.

Example

Let $R = \mathbb{Z}[x]$ be the set of all polynomials over $\mathbb{Q}$. Given $a \in \mathbb{R}$, define $\phi_a : \mathbb{Q}[x] \to \mathbb{R}$ by
$$\phi_a(f(x)) = f(a)$$

$\phi_a$ is evaluation homomorphism.

for $a = \sqrt{2}$, we have
$$ \begin{align}
\mathbb{Q}[\sqrt{2}] &:= \phi_{\sqrt{2}}(\mathbb{Q}[x]) = { f(\sqrt{2}) \mid f[x] \in \mathbb{Q} } \
&= {a_0 + a_1\sqrt{2} + a_2 (\sqrt{2})^2 + \ldots + a_n(\sqrt{2})^n \mid a_i \in \mathbb{Q}, n \in \mathbb{Z}^+ } \
&= {a + b\sqrt{2} \mid a, b \in \mathbb{Q} }
\end{align}$$

Isomorphisms

Definition

Let $R$ and $R^\prime$ be two rings. A function $\phi : R \to R^\prime$ is an isomorphism if $\phi$ is one-to-one, onto and homomorphism. Then we say $R$ and $R^\prime$ is isomorphic.

Example

Let $\alpha$ be a basis of n-dimensional vector space $V$ over $F$. Then the matrix representation ${\rm Rep}_\alpha : {\rm End}(V) \to M_n (F)$ is a ring isomorphism.

Example

For two coprime integer $n, m$, $\rho : \mathbb{Z}_{mn} \to \mathbb{Z}_n \times \mathbb{Z}_m$ is group isomorphism by given that
$$\rho(\bar a) = (\bar a, \bar a)$$
It’s clear that $\rho$ is also a multiplicative homomorphsim. Thus, $\rho$ is a ring isomorphism.

Example

The map $\phi : \mathbb{C} \to \left{ \begin{pmatrix} a & b \ -b & a \end{pmatrix} \middle| a, b \in R \right}$ defined by
$$\phi(a + bi) = \begin{pmatrix} a & -b \ b & a \end{pmatrix}$$
Show that it is a isomorphism.

Regard $\mathbb{C}$ as a vector space over $\mathbb{R}$ with the basis $\alpha = {1, i}$.

Consider the map $$\rho : \mathbb{C} \to {\rm End}(\mathbb{C})$$
given by $\rho_x(y) = xy$. It is easy to see that $\rho$ is injective ring homomorphism.

Note that for $a, b \in \mathbb{R}$
$$\rho_{a+bi}(1) = a \cdot 1 + b \cdot i \qquad \rho_{a+bi}(i) = -b \cdot 1 + a \cdot i$$
Therefore,
$${\rm Rep}\alpha(\rho{a+bi}) = \begin{pmatrix} a & -b \ b & a \end{pmatrix} = \phi(a+bi)$$
