Zero Divisor

Definition

A non-zero element \(a\) of a ring \(R\) is a zero divisor if there exists a non-zero element \(b\) such that \(ab = 0\).

Example

  1. \(\bar 2\), \(\bar 3\), \(\bar 4\) are all zero disvisors in \(\mathbb{Z}_6\).
  2. The matrices \(\begin{pmatrix} 0 & 0 \\ 1 & 0 \end{pmatrix}\) and \(\begin{pmatrix} 0 & 0 \\ 0 & 1 \end{pmatrix}\) are zero divisors in \(M_2(\mathbb{Z})\).

Theorem

The zero divisors of \(\mathbb{Z}_n\) is the the nonzero elements not relatively prime to \(n\).

Corollary

If \(p\) is a prime, then \(\mathbb{Z}_p\) has no zero divisor.

Proof

For \(\bar m \in \mathbb{Z}_n\)

  1. If \({\rm gcd}(m, n) = d > 1\). We have \[\bar m ({n \over d}) = \bar n ({\bar m \over d}) = \bar 0\] so \(\bar m\) is a zero divisor.
  2. If \({\rm gcd}(m, n) = 1\).
    We assume that \(\bar m \bar k = 0\) \(\implies\) \(n \mid mk\)
    Since \(m\) is relatively prime to \(n\), we have \(n \mid k\) \(\implies\) \(\bar k = \bar 0\)
    We see that \(\bar m\) is not a zero divisor.


Theorem - Cancellation Law

The cancellation law holds if and only if \(R\) has no zero divisors

Remark

  1. For a ring with zero divisor, even though the cancellation law not hold in general, but still holds in some cases that a multiplicative inverse exists.
    1. In \(\mathbb{Z}_5\), \(2b = 2c\) implies \(b = c\) because \(2\) is relatively prime to \(15\).
    2. \(A\) is an invertible matrix in \(M_2(\mathbb{R})\), then \(AB = AC\) implies \(B = C\).
  2. If \(R\) is a ring with no zero divisor, then the equation \(ax = b\) has at most one solution in \(R\).
Proof
  1. \(\implies\)
    Assume that the cancellation holds but \(a \neq 0\) is a zero divisor in \(R\).
    Then we have \(ab = 0\) for some \(b \neq 0 \in R\).
    \(ab = 0 = a0\) but \(b \neq 0\), which is a contradiction.
  2. \(\impliedby\)
    Assume that \(R\) has no zero divisor. If \(ab = ac\) and \(a \neq 0\), then \(ab - ac = 0\).
    By distributive law, \(a(b - c) = 0\).
    Since \(R\) has no zero divisor and \(a \neq 0\), we can get \(b - c = 0 \implies b = c\).


Notation - b/a

When a ring R is commutative, having unity and no zero divisors. we can denote \(a^{-1}b\) as \(b/a\). And \(1/a\) deonte \(a^{-1}\).

But when R is not communitive, we can't use this notation since we don't know \(b/a\) means \(a^{-1}b\) or \(ba^{-1}\).

Integral Domains

Definition

A commutative ring \(R\) with unity \(1 \neq 0\) and has no zero divisors is an integral domain.

Example

  1. The ring of integres \(\mathbb{Z}\) is an integral domain. (That's why integral domain is called "integral" domain)
  2. If \(p\) is a prime, then \(\mathbb{Z}_p\) is an integral domain.
  3. The direct product of two nonzero rings \(R, S\) is not an integral domain since \((r, 0)(0, s) = (0, 0)\) for all \(r \in R\) and \(s \in S\).

Finite Integral Domains

Theorem

Every finite integral domain is a field.

Corollary

If \(p\) is a prime, then \(\mathbb{Z}\) is field.

Remark - Wedderburn's theorem

Every finite division ring is a field.

Proof

Let \(D\) be a finite integral domain.
Since field is a communicative division ring, to show \(D\) is a field, we need to show that every nonzero element in \(D\) is unit.
Let \(0, 1, a_1, a_2, \dots, a_n\) be all the elements of \(D\).
The products \(0 = a0, a = a1, aa_1, \dots, aa_n\) are distinct since the cancellation law.
Then the products must be all elements of \(D\), so that there must be some \(a_i\) s.t. \(aa_i = 1\).
Therefore, for all \(a \in D\) is unit. This prove the theorem.

Characteristic of Ring

Definition

Let R be a ring, the least interger \(n\) such that \(n \cdot a = 0\) for all \(a \in R\) is the characteristic of R.
If no such positive integer exist, then \(R\) is of characteristic \(0\).

Example

  1. The rings \(\mathbb{Z}_n\) are of characteristic \(n\).
  2. The rings \(\mathbb{Z},\, \mathbb{Q},\, \mathbb{R},\, \mathbb{C}\) are all of characteristic \(0\).

Theorem

Let \(R\) be a ring with unity.
If \(n \cdot 1 \neq 0\) for all \(n \in \mathbb{Z}^{+}\), then \(R\) has characteristic \(0\).
If \(n \cdot 1 = 0\) for some \(n \in \mathbb{Z}^{+}\), then the smallest such \(n\) is the characteristic of \(R\).

Proof

If \(n \cdot 1 \neq 0\) for all \(n \in \mathbb{Z}^{+}\), then \(R\) is clearly of characteristic 0 by definition.
If \(n \cdot 1 = 0\) for some \(n \in \mathbb{Z}^{+}\), then for all \(a \in R\), we have \[n \cdot a = (a + \dots + a) = a (1 + \dots + 1) = a(n \cdot 1) = a0 = 0\] which prove the theorem.


Theorem

Let \(R\) be a ring with unity of characteristic \(n\) (\(n\) can be zero). Then \(\langle 1_R \rangle \cong \mathbb{Z}_n\).

Remark

\(\mathbb{Z}_n\) can be regard as the quotient ring \(\mathbb{Z} / n\mathbb{Z}\).
Therefore, we have \(\mathbb{Z} / 0\mathbb{Z} = \mathbb{Z}\) when \(n = 0\). So that we can difine the characteristic in this case to be zero instead of infinity.

Corollary

Every ring with unity contains \(\mathbb{Z}\) or \(\mathbb{Z}_n\) as a subring.

Proof

For \(n = 0\), the order of \(1_R\) is infinite. That is \(\langle 1_r \rangle \cong \mathbb{Z}\)
For \(n \neq 0\), we can write \(\langle 1_R \rangle\) as \(\{m 1_r \mid m \in \mathbb{Z}_n \}\), since \(1_R \times n = 1_R \times 0 = 0\)
Let \(\phi : \langle 1_R \rangle \to \mathbb{Z}_n\), defined by \[\phi(m1_R) = m\] Now we need to show \(\phi\) is a isomorphism:

  1. Suppose \(\phi(a) = \phi(b)\), then \(a = b = m1_R\) where \(m \in \mathbb{Z}_n\)
    \(\phi\) is one-to-one
  2. For all \(a \in \mathbb{Z}_n\), \(\exists a1_R \in \langle 1_R \rangle\) such that \(\phi(a1_R) = a\)
    \(\phi\) is onto
  3. For all \(x1_R, y1_R \in \langle 1_R \rangle\), where \(x, y \in \mathbb{Z}_n\) \[\phi(x1_R) + \phi(y1_R) = x + y = \phi(a + b)\] \[\phi(x1_R)\phi(y1_R) = xy = \phi(ab)\] \(\phi\) is homomorphism.

Therefore, \(\phi\) is isomorphism. And we can know that \(\langle 1_R \rangle \cong \mathbb{Z}_n\)
