Total Probability Theorem

if the events \(A_1, A_2, \ldots , A_n\) forms a partition of the sample space, then \[P(x) = \sum_{i = 0}^{n} {P(x | A_i)}\] \[f_X(x) = \sum_{i = 0}^{n} {P_{A_i} f_{x | A_i}(x)}\]

Normal Random Variables

Also called Gaussian Random Variable

PDF

\[ f(x) = {1 \over \sqrt{2\pi}\sigma} e^{- (x-\mu)^2 / 2 {\sigma}^2} \]

Standard Normal Random Variable

A normal variable Y is called Standard Normal Random Variable, if it has mean \(0\) and unit variance. It's CDF is donote by \(\Phi\)

\[ \Phi (y) = P(Y \leq y) = P(Y < y) = {1 \over \sqrt{2\pi}} \int_{-\infty}^{y} {e^{-{t^2 \over 2}} {\rm d}x} \]

\(\Phi(y) = 1 - \Phi(-y)\) for all \(y\).

Standardize

\(X\) is a normal random variable with mean \(\mu\) and variance \(\sigma^2\).
We can standardize \(X\) by defining \(Y = {X - \mu \over \sigma^2}\) \(Y\) is normal with mean 0, unit variance. After standardizing, we can calculate CDF of \(X\) by using \(Y\) and table.

Exponential Random Variables

PDF

\[f_X(x)= \left\{ \begin{matrix} & \lambda e^{-\lambda x} & \text{if } x \geq 0\\ & 0 \hfill & \text{else}\\ \end{matrix} \right. \] where \(\lambda\) is a positive parameter characterizing the PDF.

For any \(a \geq 0\), we have \[P(X \geq a) = \int_{a}^{\infty} {\lambda e^{-\lambda x}} \, {\rm d}x = \left. e^{-\lambda x} \right|_a^{\infty} = 1\]

Expection

\[E[X] = \frac{1}{\lambda}\]

Verify: \[ \begin{eqnarray} E[X] &=& \int_{0}^{\infty} {x \lambda e^{-\lambda x}} \, {\rm d}x \\ &=& \left. (-xe^{-\lambda x}) \right|_{0}^{\infty} + \int_{0}^{\infty} {e^{\lambda x}} \, {\rm d}x \\ &=& 0 - \left. {e^{-\lambda x}}\over{\lambda} \right|_{0}^{\infty} \\ &=& {1}\over{\lambda} \\ \end{eqnarray} \]

Variance

\[Var(X) = {1}\over{\lambda^2}\]

Verify: \[ \begin{eqnarray} E[X^2] &=& \int_{0}^{\infty} {x^2\lambda e^{-\lambda x}} \, {\rm d}x \\ &=& \left. (-x^2 e^{-\lambda x}) \right|_{0}^{\infty} + \int_{0}^{\infty}{2xe^{\lambda x}} \, {\rm d}x \\ &=& 0 + {2}\over{\lambda} E[X] \\ &=& {2}\over{\lambda^2} \\ \end{eqnarray} \] using the formula \(var(X) = E[X^2] - (E[X^2])^2\). We have \[var(X) = \frac{2}{\lambda^2} - \frac{1}{\lambda^2} = \frac{1}{\lambda^2}\]

CDF

Memoryless

The time of buld is light until burn out is a exponential random variable with parameter \(\lambda\) A buld has lighting up for \(t\) times unit. \(X\) is the additional time until the light bulb burns out. What is the conditional CDF of X. given the event A?

\[ \begin{align} P(X \geq x|A) &= P(T > t + x | T > t) = \frac{P(T > t + x \text{ and } T > t)}{P(T > t)} \\ &= \frac{P(T > t + x)}{P(T > t)} = \frac{e^{-\lambda (t + x)}}{e^{-\lambda t}} = e^{-\lambda x} \end{align} \]

MLE

Let \(X_1, \ldots, X_n\) be a random sample with the PDF \(\theta e^{-\theta x}\), find the MLE of \(\theta\)

  1. likehood function
    \(L(\theta) = \prod P(X) = \theta^n e^{\theta \sum {X_i}}\)
  2. 取 log
    \(\ln L(\theta) = n \ln \theta - \theta \sum {X_i}\)
  3. 微分找最大
    \(\dfrac{\mathrm{d}}{\mathrm{d}\theta} \ln L(\theta) = \dfrac{n}{\theta} - \sum {X_i} = 0\)
    \(\implies \theta = \dfrac{n}{\sum {X_i}}\)
