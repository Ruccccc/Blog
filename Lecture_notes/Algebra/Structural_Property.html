Structural Property

A structural property of a binary strucuture is one that must be shared by any isomorphic structure.

**Example**

Structural properties

  1. The set has 4 elements.
  2. The operation is commutative.
  3. $ x * x = x$ for all $x \in S$
  4. The equation $a * x = b$ has solution in $S$ for all $a, b \in S$

non-structural properties

  1. The set $S$ is a subset of $\mathbb{C}$
  2. The number 4 is an element.
  3. The operation is called “addition”.
  4. The elements of $S$ are matrices.

Identity Element

Definition

Let $(S,*)$ be a binary structure. An element $e$ of $S$ is an identity element for $\ast$ if $e * s = s * e = s\ \forall\ s\in\ S$

**Example**
  1. In $\left( \mathbb{Z} ,\ +\right)$, the element 0 is an identity element.
  2. In $\left( \mathbb{Z} ,\ *\right)$, the element 0 is an identity element.
  3. In $\left( \mathbb{Z}_{n},\ *\right)$, the element $\overline{1}$ is an identity element.
  4. In $\left(M_2 \left( \mathbb{R} \right),\ +\right)$, the zero matrix is an identity elements.

Theorem – Uniqueness

A binary structure $\left( S,\ *\right)$ has at moest one identity element.

**Proof**

Suppose That $e$ and $e^\prime$ are both identity elements.
Consider $e * e^{\prime}$.
Since $e$ is identity element, we have
$$e * e^{\prime} = e^{\prime}$$
On the other hand, $e^{\prime}$ is an identity element, we also have
$$e * e^{\prime} = e$$
We conclude the $e=e^{\prime}$


Theorem

Suppose that $(S, *)$ has an identity element $e$. If $\phi : S \mapsto S^\prime$ is an isomorphism of $(S, *)$ with $(S^\prime, *^\prime)$, then $\phi(e)$ is an identity element for $\ast^\prime$.

In other words, having an identity element is a structure property.

Proof

We need to show that
$$\phi(e) *^\prime s^\prime = s^\prime \ast^\prime \phi(e) = s^\prime$$
for all $s^\prime \in S^\prime$
Since $\phi$ is isomorphism, $\phi$ is onto.
Thus, there exists $s \in S$ s.t. $\phi(s) = s^\prime$
Then
$$\phi(e) *^\prime s^\prime = \phi(e) *^\prime \phi(s)$$
By $\phi$ is isomorphism again, we have
$$\phi(e) *^\prime \phi(s) = \phi(e \ast s) $$
It follows that
$$\phi(e) *^\prime s^\prime = \phi(e \ast s) = \phi(s) = s^\prime$$
By same token, we can also show that $s^\prime *^\prime \phi(e) = s^\prime$.
We can conclude that $\phi(e)$ is an identity element.
