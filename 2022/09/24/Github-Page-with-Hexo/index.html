Github Page with Hexo

Posted on 2022-09-24 Edited on 2022-09-25

A note about how I create this page.

Install Hexo

requrements

  • git
    mac : brew install git
  • node.js
    mac : brew install node

install Hexo with npm

npm install -g hexo-cli

Setup

Initialize

hexo init [folder]
  • _config.yml
    Site configuration file.
  • package.json
    Application data.

Change theme

I use NexT as the theme of this site.

download

  • though npm

    npm install hexo-theme-next

  • though git

    git clone https://github.com/next-theme/hexo-theme-next themes/next

Set theme

edit _config.yml

theme: next

configurations

Update

npm install hexo-theme-next@latest

Add new post

hexo new [layout] <title>

Deploy with Github Page

Github

  1. create a repository named <username>.github.io
  2. In Settings > page set branch to master(or gh-pages)

Local

  1. Edit _config.yml
    # Deployment
## Docs: https://hexo.io/docs/one-command-deployment
deploy:
  type: git
  repo: https://github.com/<username>/<username>.github.io.git
  branch: master
  message:

Deploy

  1. clean the cache file and generated files.

    hexo clean
hexo cl        # short version

  2. gernerate static files.

    hexo gernerate
hexo g         # short version

  3. deploy the website.

    hexo deploy
hexo d         # short version
