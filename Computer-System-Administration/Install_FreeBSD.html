Install FreeBSD

Install FreeBSD

Before install

  • Virual Machine
    • VMware, VirtualBox, etc...
  • Image file
    • https://www.freebsd.org/releases/
    • Using Release 13.1 for this course.

Stuck in "UEFI Interactive Shell"

type exit, then negivate to boot Manager. After all, boot VM with your image file.

Starting

  • set almost every default.

Sudo

Install

  1. switch to root by using su -
  2. install package sudo

Allowing user to execute sudo

  1. use visudo to edit sudoer file
  2. save and logout
  3. type sudo whoami to check if it works.

Excute sudo without password

  1. use visudo to edit sudoer file
  2. save and logout

Add user

  1. switch to root
  2. use assuser
  3. type in everything asked

Update FreeBSD

  1. fetch new version
  2. install fetch
  3. reboot and check version

Using port

Port collection is in /usr/ports by default.

Before using port to install application, we need to fetch port collection first.

Fetch Port Collection

By Portsnap

  1. Clone snapshot of port collection to /var/db/portsnap
  2. extract snapshot to /usr/ports
  3. Update /usr/ports
