Introduction

Graph

A graph \(G = (V, E)\) consists two sets: \(V(G)\) and \(E(G)\), represent vertex and edge respectively. The number of vetices in a graph \(G\) is denoted \(|V(G)|\) or \(|G|\).

Subgraph

Definition

Graph \(H\) is gubgraph of graph \(G=\) if \(V(H) \subseteq V(G)\) and \(E(H) \subseteq E(G)\).
\(H_1\) is a induced subgraph of \(G\) if \(V(H_1) \subseteq V(G)\) and every availiable edge from \(G\) between the verieces in \(V(H_1)\) is included.
\(H_2\) is a spanning subgraph of \(G\) if \(V(H_2) = V(G)\) and \(E(H_2) \subseteq E(G)\)

Directed graph

Definition

A directed graph or digraph, is a graph \(G=(V,A)\), where \(V(G)\) is a vertex set and A(G) is an arc set. An arc is an ordered pair of vertices. An arc \(xy\), the head is the starting vertex \(x\) and the tail is the ending vertex \(y\). The indegree of a vertex \(x\) is the number of arcs which \(x\) is a tail, denoted \(deg^{-}(x)\) The outdegree of a vertex \(x\) is the number of arcs which \(x\) is a head, denoted \(deg^{+}(x)\) The underlying graph for a digraph is the graph that removing the direction.

Weighted Graphs

Definition

A weighted graph is a graph that each edges has real number associated with it. This number refferred to as the weight and denoted \(w(xy)\) for the edge \(xy\). (Weight can also be assigned to vertices)

Complete Graph

Definition

A simple graph \(G\) is complete if every pair of distinct vertice is adjacent. The complete graph on \(n\) vertices is denoted \(K_n\).

Properties

  1. Each vertex in \(K_n\) has degree \(n-1\)
  2. \(K_n\) has \(n(n-1) \over 2\) edges.
  3. \(K_n\) contains the most edges out of all simple graphs on \(n\) vertices.

Definition

The clique-size of a graph is the largest integer n such that \(K_n\) is a subgraph of \(G\) but \(K_{n+1}\) is not.

Graph Complement

Definition

Given a simple graph \(G = (V, E)\), define the complement of \(G\) as the graph \(\bar G = (V, \bar E)\), where an edge \(xy \in \bar E\) if and only if \(xy \notin E\)

Bipartite Graph

Definition

A graph \(G\) is bipartitie if the vertices can be partitioned into two sets \(X\) and \(Y\) so that every edge has one endpoint in \(X\) and the other in \(Y\).
\(K_{m,n}\) is the complete bipartite graph where \(|X| = m\) and \(|Y| = n\) and every vertex in \(X\) is adjacent to every vertex in \(Y\).

Degree Sequence

Definition

The degree sequence of a graph is a listion of the degrees of the vertices. It is customary to write in decreasing order. If a sequence is degree sequence of a simple graph then we call it graphical

Havel-Hakimi Theorem

Theorem

An non-increasing \(S: s_1, s_2, \ldots, s_n\) (for \(n \geq 2\)) of nonnegative integers is graphical if and only if the sequence \(S^\prime: s_{2}-1, s_{3}-1, \ldots, s_{s_{1}+1}-1, s_{s_{1}+2}, \ldots, s_n\) is graphical.
