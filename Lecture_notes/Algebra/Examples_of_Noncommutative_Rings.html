The Ring of Endomorphisms

Definition

Let \(G\) be a group. A homomorphism from \(G\) to itself is an endomorphism.

Theorem

Let \((G, +)\) be a abelian group. The set \({\rm End}(G)\) of all endomorphism of \(G\) forms a ring with the function addiction and composition defined by \(\phi + \psi : g \mapsto \phi(g) + \psi(g)\) and \(\phi \circ \psi : g \mapsto \phi(\psi(g))\)

Hamilton Quaternion

We will show that the Hamilton quaternion algebra is a noncommutative division ring.

Let \(\Bbb{H}\) be the set \(\Bbb{R}^4\) endowed with the usual component-wise addition.
We write \(1 = (1, 0, 0, 0)\), \(i = (0, 1, 0, 0)\), \(j = (0, 0, 1, 0)\), \(k = (0, 0, 0, 1)\), so that \((a_1, a_2, a_3, a_4) = a_1 + a_2 i + a_3 j + a_4 k\)
We defined the multiplication on \(\Bbb{H}\) by setting \[1a = a1 = a \forall a \in \Bbb{H}\] \[i^2 = j^2 = k^2 = -1\] \[ij = k, jk = i, ki = j, ji = -k, kj = -i, ik = -j\]

Define the multiplication for general elements by \[\begin{align} (a + bi + cj + dk)(w + xi + yj + zk) & = (aw - bx - cy - dz) \\ & + (ax + bw + cz - dy)i \\ & + (ay + cw - bz + dx)j \\ & + (az + dw + by - cx)k \\ \end{align} \] Then the distributive laws and the associativity of multiplication will hold.

Now we show that \(\Bbb{H}\) is a division ring, i.e., every nonzero element in \(\Bbb{H}\) has a multiple inverse.
Let \(x = (a + bi + cj + dk)\)
Observe that \((a + bi + cj + dk)(a - bi - cj - dk) = a^2 + b^2 + c^2 + d^2\), which is not zero if \((a + bi + cj + dk) \neq 0\).
Thus, writing \(|x|^2 = a^2 + b^2 + c^2 + d^2\) the inverse of \(x\) is \[\frac{|x|^2}{a} - \frac{|x|^2}{b}i - \frac{|x|^2}{c}j - \frac{|x|^2}{d}k\] This shows that every nonzero element in \(\Bbb{H}\) is a unit. Therefore \(\Bbb{H}\) is a non-commutative ring.

Algebra

Defintion

An algebra is a vector space over a field with an multiplication structure, or equivalently, a ring with a vector space structure.

Theorem

\(\Bbb{R}, \Bbb{C}, \Bbb{H}\) are the only division algebras over \(\Bbb{R}\)

Remark

The Hamilton quaternion is the only non-commutative division algebra over \(\Bbb{R}\).
