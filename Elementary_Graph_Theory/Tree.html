Trees

Spanning Trees

Definition

A graph is acyclic is there are no cycles or circuits. A graph is a tree if it's acyclic and connected. A graph is a froest if it's an acyclic graph.

Definition

A spanning tree is a spanning subgraph that is a tree. Given a weighted graph \(G\), \(T\) is a minimum spanning tree of \(G\) if it is a spanning tree with the least total weight.

Kruskal's Algorithm

Finding a maximum/minimum spanning tree for a connected graph, a forest for nonconnected graph

  • steps
    1. Choose the least weight edge and add it to \(T=(V, E^\prime)\)
    2. Repeat Step 1. as long as no circuit is created.

Prim's Algorithm

Finding a maximum/minimum spanning tree for a connected graph.

  • Steps
    1. Choose root \(v\), highlight it and add it to \(T=(V^\prime, E^\prime)\)
    2. Let \(S\) be the set of all edges with exactly endpoint from \(V(T)\). Choose the least weight edge in \(S\), highlight it and add it to \(T\).
    3. Repeat step 2. until \(V(T) = V(G)\).

Tree Properties

Theorem

Every tree with at least two vertices has a leaf.

  • Proof by contradiction. Assume that there is no leaf.

Lemma

Removing a leaf \(v\) on a tree \(T\), the graph \(T - v\) is still a tree.

Theorem

A tree with \(n\) vertices has \(n-1\) edges for all \(n \geq 1\).

  • Proof by induction.

Corollary

The total degree of a tree on \(n\) vertices is \(2n-2\).
