Homomorphism

Homomorphism

Definition

A map \(\phi\) of a group \(G\) into a group \(G^\prime\) is a homomoriphism if \[\phi(ab) = \phi(a) \phi(b)\] for all \(a, b \in G\).

Definition

Let \(\phi\) be a mapping of a set \(X\) into a set \(Y\). Let \(A \subset X\) and \(B \subset Y\).

  1. The image of \(A\) under \(\phi\) is the set \(\{ \phi(a) : a \in A \}\), denoted by \(\phi(A)\).
  2. The set \(\phi(X)\) is the range of \(\phi\)
  3. The inverse image (or preimage) of \(B\) in \(X\) is \(\{x \in X : \phi(x) \in B\}\), denoted by \(\phi^{-1}(B)\).

Theorem

Let \(\phi\) be a homomorphism of a group \(G\) into a group \(G^\prime\).

  1. \(\phi(e) = e^\prime\)
  2. \(\phi(a^{-1}) = \phi(a)^{-1}\) for all \(a \in G\).
  3. If \(H\) is a subgroup of \(G\), then \(\phi(H)\) is a subgroup of \(G^\prime\).
  4. If \(H^\prime\) is a subgroup of \(G^\prime\), then \(\phi^{-1}(H^\prime)\) is a subgroup of \(G\).
Proof

Corollary

Let \(\phi : G \to G^\prime\) be a group homomorphism. If \(\phi\) is one-to-one then \(G\) and \(\phi(G)\) are isomorphic.

Kernel

Definition

The subgroup \(\phi^{-1}(\{ e^\prime \})\) is called the kernel of \(\phi\) denoted by \({\rm ker}(\phi)\).

Example

Recall that det:\(GL(n, \mathbb{R}) \to \mathbb{R}^\prime\) is a group homorphism. Therefore, \[SL(n, \mathbb{R}) = \{ A \in GL(n, \mathbb{R}) \mid det(A) = 1 \} = \text{ker det}\] is a subgroup (also called the special linear group)

Problem

Let \(\phi : G \to G^\prime\) be a group homomorphism. For \(a \in G\), is there any relation between the order of \(a\) and the order of \(\phi(a)\)?

Suppose \({\rm ord}(a) = n < \infty\) then \[e^\prime = \phi(e) = \phi(a^n) = \phi(a)^n\] Therefore \({\rm ord}(\phi(a)) | n\).

Theorem

Let \(\phi : G \to G^\prime\) be a group homomorphism, and let \(H = {\rm ket}(\phi)\).
Let \(a \in G\). Then the set \[\{ x \in G : \phi(x) = \phi(a) \} = \phi^{-1}(\{\phi(a)\})\] is the left coset \(aH\), and also the right coset \(Ha\). Consequently, the partition of \(G\) intot left cosets is the same as the parrtition into right coset.

Proof

To show \(\phi^{-1}(\{\phi(a)\}) = aH\), we will show that

  1. \(aH \subset \phi^{-1}(\{\phi(a)\})\): Let \(x = ah \in aH\). Since \(\phi\) is a homomorphism, we have \(\phi(x) = \phi(a)\phi(h) = \phi(a)e^\prime = \phi(a)\). Therefore, \(x \in \phi^{-1}(\{\phi(a)\})\).
  2. \(\phi^{-1}(\{\phi(a)\}) \subset aH\): Suppose that \(x \in \phi^{-1}(\{\phi(a)\})\). We have \[\phi(x) = \phi(a)\] \[e^\prime = \phi(a)^{-1}\phi(x) = \phi(a^{-1}\phi(x) = \phi(a^{-1}x)\] Thus, \(a^{-1}x \in H\). We see that \(s \in aH\).

We can prove \(\phi^{-1}(\{\phi(a)\}) = Ha\) by similar method.

Corollary

A group homomorphism \(\phi : G \to G^\prime\) is one-to-one if and only if \({\rm ker}(\phi) = \{e\}\).

Example

Let \(n\) be a positive integer. Let \(\phi : \mathbb{Z} \to \mathbb{Z}_n\) be defined by \(\phi(a) = \bar a\), the residue class modulo \(n\) containing \(a\). then \({\rm ker}(\phi) = n\mathbb{Z}\). Also, for each \(\bar b \in \mathbb{Z}_n\), the set \(\phi^{-1}(\{\bar b\}) = \{\ldots, b-2n, b-n, b, b+n, b+2n, \ldots \} = b + n\mathbb{Z}\), which is indeed a coset.

Normal Subgroup

Definiton

A subgroup \(H\) of a group \(G\) is normal if its left cosets and right cosets coincide, that is \[\begin{array}{} gH = Hg & \text{ for all } g \in G \\ \end{array}\]

Note that all subgroups of abelian groups are normal.

We use the notation \(H \triangleleft G\) ot denote that \(H\) is a normal subgroup of \(G\).

Corollary

If \(\phi : G \to G^\prime\) is a group homomorphism, then \({\rm ker}(\phi)\) is a normal subgroup.

Proposition

Let \(N\) be a subgroup of index two of \(G\). Then \(N\) is normal.

Proof

For \(g \in G\), if \(g \in N\), then \(gN = N = Ng\). If \(g \notin N\), then \(G = N \cup gN\). Therefore. \(gN = G \backslash N\). By same token, we also have \(Ng = G \backslash N\), which implies \(gN = Ng\). Therefore, \(N\) is normal.

Theorem

The following are equivalent conditions for a subgroup \(H\) of a group \(G\) to be normal.

  1. \(ghg^{-1} \in H\) for all \(g \in G\) and all \(h \in H\).
  2. \(gHg^{-1} \in H\) for all \(g \in G\).
  3. \(gH = Hg\) for all \(g \in G\).
  4. \(aHbH = abH\) for all \(a, b \in G\).

Theorem

Let \(H\) be a subgroup of a group \(G\). Suppose \(G\) is generated by \(g_1, \cdots, g_n\). Then if \(g_iHg_i^{-1} = H\) for all \(i\), then \(H\) is normal.

Quotient Groups

Theorem

\((G/N , \ast)\) is a group, called the quotient group of \(G\) by \(N\).

Example

Let \(G = D_4\) and \(N = \langle \sigma^2 \rangle\). Determine the structure of \(G/N\).

\(G = N \cup \sigma N \cup \tau N \cup \tau\sigma N\)
\(G/N\) is a group of order 4. Since \(\sigma N = \{\sigma, \sigma^3\} = N\sigma\) and \(\tau N = \{\tau, \tau\sigma^2\} = N \tau\),
N is normal. Therefore, \(G/N \cong \mathbb{Z}_4 or \mathbb{Z}_2 \times \mathbb{Z}_2\).
Note that \((\tau N)^2 = (\sigma N)^2 = (\tau\sigma N)^2 = N\).
Therefore, \(G/N \cong \mathbb{Z}_2 \times \mathbb{Z}_2\).

Homomorphisms and Normal Subgroups

Theorem

Let \(N\) be a normal subgroup of \(G\). Then \(\gamma : G \to G\backslash N\) goven by \(\gamma(g) = gN\) is a homomorphism with the kernel N.

Theorem - The first isomorphism theorem

Let \(\psi : G \to G^\prime\) be a group homomorphism with kernel \(N\). Then \(G/N\) and \(\psi(G)\) is isomorphic. The canonical isomorphism \(\mu : G/N \to \psi(G)\) is given by \(\mu(gN) = \psi(g)\).

Theroem

Every abelian group with \(n\) generators is isomorphic to a quotient group of \(\mathbb{Z}^n\)

Quotient Group Computation

Problem

Given a quotient group \(G/H\), find an isomorphic group \(G^\prime\) that is more familiar to us. (So that to study the structure of \(G/H\), we can work on the more familiar group \(G^\prime\) instead).

Some simple facts

  1. \(G/{e}\) is isomorphic to \(G\) itself.
  2. \(G/G\) is isomorphic to the trivial group with one element.
  3. If \(G\) is cyclic, then \(G/H\) is also cyclic.
  4. If \(G = G_1 \times G_2\), then \(G/({e_1} \times G_2)\) is isomorphic to \(G_1\).
  5. If \(G = G_1 \times \G_2\), \(H_1 \triangleleft G_1\), and \(H_2 \triangleleft G_2\), then \(G / (H_1 \times H_2)\) is isomorphic to \((G_1/H_1) \times (G_2/H_2)\)

Example

Classify the quotient group \(\mathbb{Z}_4 \times \mathbb{Z}_2 / \langle (\bar 2, \bar 0) \rangle\).

soluation 1

Write \(G = \mathbb{Z}_4 \times \mathbb{Z}_2\) and \(H = \{(\bar 0, \bar 0), (\bar 2, \bar 0)\}\).
The cosets are:
\((\bar 0, \bar 0) + H = \{(\bar 0, \bar 0), (\bar 2, \bar 0)\}\), \((\bar 1, \bar 0) + H = \{(\bar 1, \bar 0), (\bar 3, \bar 0)\}\), \((\bar 0, \bar 1) + H = \{(\bar 0, \bar 1), (\bar 2, \bar 1)\}\) and \((\bar 1, \bar 1) + H = \{(\bar 1, \bar 1), (\bar 3, \bar 1)\}\)
Since \(G/H\) is abelian of order \(4\), it is isomorphic to \(\mathbb{Z}_4\) ro \(\mathbb{Z}_2 \times \mathbb{Z}_2\).
Now, \(2((\bar 1, \bar 0) + H) = (\bar 2, \bar 0) + H = H\), \(2((\bar 0, \bar 1) + H) = (\bar 0, \bar 0) + H = H\),
\(2((\bar 1, \bar 1) + H) = (\bar 2, \bar 0) + H = H\).
There is no element of order \(4\) in the group \(G/H\), so \(G/H \cong \mathbb{Z}_2 \times \mathbb{Z}_2\).

soluation 2

Define a function \(\psi : \mathbb{Z}_4 \times \mathbb{Z}_2 \to \mathbb{Z}_2 \times \mathbb{Z}_2\) by \[\psi(a \text{ mod } 4, b \text{ mod } 2) = (a \text{ mod } 2, b \text{ mod } 2)\] which is a well-defined since \(2|4\).
It's easy to see that \(\psi\) is a homomorphism. The kernel is \(\{(\bar 0, \bar 0)\} = H\). Moreover, \(\psi\) is onto(prove by check one by one).
Therefore, by the first isomorphism theorem we have \(\mathbb{Z}_4 \times \mathbb{Z}_2 / \langle (2, 0) \rangle \cong \mathbb{Z}_2 \times \mathbb{Z}_2\).
